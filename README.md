# bun-typescript-docker-starter

To install dependencies:

```bash
bun install
```

To develop:

```bash
bun run dev
```

To run checks:

```bash
bun run fmt:check
bun run lint
```

To format files:

```bash
bun run fmt
```

This project was created using `bun init` in bun v1.2.10. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
